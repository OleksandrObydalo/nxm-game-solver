Описание метода

Программа решает задачу поиска оптимальных смешанных стратегий в матричной игре с помощью линейного программирования.

Для игрока A: Максимизация минимального гарантированного выигрыша

Для игрока B: Минимизация максимального проигрыша

Используется алгоритм симплекс-метода для решения задач линейного программирования.
